배열은 아이템에 대한 식별자로 숫자사용. 데이터가 추가되면 배열전체에서 중복되지 않는 인덱스가 자동으로 만들어져서 추가된 데이터에 대한 식별자가 됨
이 인덱스를 이용해서 데이터를 가저오게 되는것. 만약 인덱스로 문자를 사용하고 싶다면 객체를 사용해야함

객체 생성

// 방법1
let grades = {'key': value, 'key': value, 'key': value, ...};
let grades = {'egoing': 10, 'k8805': 6, 'sorialgi': 80};
// 방법2
let grades = {}; // {} 대신해서 new object() 사용할수 있음
grades['egoing'] = 10;
grades['k8805'] = 6;
grades['sorialgi'] = 80;
