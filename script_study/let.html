변수

변수(variavle)는 (문자나 숫자 같은) 값을 담는 컨테이너로 값을 유지할 필요가 있을떄 사용
여기 담겨진 값은 다른값으로 바꿀수 있음
변수는 마치 자연어에서 대명사와 비슷한 역할

let - 블록 레벨 변수를 선언, "변수가 선언된 블록"
const - 변경되지 않을 변수를 선언, "변수가 선언된 블록"
var - "변수가 선언된 함수"에서 사용 (현재 기준 사용 자제..)

let a = 1;
alert(a+1); //2

let a = 2;
alert(a+1); //3

let first = "coding";
alert(first+" everybody");

//변수 a에 coding ,변수 b에 everybody를 활당하는 방법은 아래와 같다.
let a = 'coding', b = 'everybody';
alert(a);
alert(b);

**변수 선언이후에 값을 바꿀때는 let을 또 쓰지 않아도 괜찮음

변수의 효용

//변수를 쓰기 전
alert(100+10);
alert((100+10)/10);
alert(((100+10)/10)-10);
alert((((100+10)/10)-10)*10);

// 변수 사용 후
let a = 100;
a = a + 10;
alert(a);
a = a / 10;
alert(a);
a = a - 10;
alert(a);
a = a * 10;
alert(a);
